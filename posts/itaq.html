ITAQ: Image Tag Recommendation Framework for Aquatic Species Integrating Semantic Intelligence via Knowledge Graphs

The ITAQ framework responds to the growing demand for advanced social image tagging in the Web 3.0 era. Specifically designed for tagging aquatic species images, it employs knowledge-centric methods aligned with semantic web standards. The framework integrates strategic knowledge curation through topic modeling, metadata generation, classification, ontology integration, and enrichment using knowledge graphs. It utilizes a balanced approach with lightweight machine learning through AdaBoost and CNN classifiers. Semantic reasoning is enhanced through context trees, KL divergence, and Second Order Co-occurrence PMI. The system's efficiency is further optimized by leveraging Ant Lion optimization for computing the most optimal solutions.
